LiteFlow在执行每一条流程链后会打印步骤，这个步骤是程序实际执行的顺序

## 普通打印

普通打印只会打印nodeId，样例如下：

```
a<100>==>c<10>==>m<0>==>q<200>==>p<300>==>p1<0>==>g<305>
```

这里的表达形式为：`组件ID<耗时毫秒>`

## 别名打印

如果你希望在打印流程链的时候增加别名描述，那你需要在定义组件的时候设置name属性，具体请参照[组件别名](/pages/4c74f0/)。

最终样例为：
```
a[组件A]<100>==>b[组件B]<0>==>m[组件M]<256>
```

这里的表达形式为：`组件ID[组件别名]<耗时毫秒>`
