如果你要依次执行a,b,c,d四个组件，你可以用`THEN`关键字，需要注意的是，`THEN`必须大写。

```xml
<chain name="chain1">
    THEN(a, b, c, d)
</chain>
```

当然你如果嵌套来写，也是和上面等价的
```xml
<chain name="chain1">
    THEN(a, b, THEN(c, d))
</chain>
```
